Paperless GMail API

Diese Anwendung wird ausschließlich für den privaten Gebrauch betrieben.

Datenverarbeitung

  • Es werden nur E-Mails aus dem eigenen Gmail-Konto abgerufen
  • Der Zugriff erfolgt ausschließlich lesend (gmail.readonly)
  • Keine Daten werden an Dritte weitergegeben
  • Alle Daten verbleiben auf dem privaten Server
